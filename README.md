## AI Helper (CLI)

Python으로 만든 아주 단순한 CLI 비서입니다.

**목표**
- 창의성이 거의 필요 없고
- 실수하면 스트레스만 생기고
- 하지만 매번 시간을 잡아먹는

그런 루틴 작업들을 빠르게 떠올리도록 돕는 것이 목적입니다.

### 버전 상태

- 지금은 **아주 초기 버전(실험용 프로토타입)** 입니다.
- 앞으로 실제 생활 속에서 조금씩 쓰면서,
  - 자주 까먹는 일,
  - 실수하면 크게 손해 보는 일,
  - AI·스크립트로 맡기고 싶은 일
  들을 계속 발견해 나갈 예정입니다.
- 목표는, 시간이 지날수록 이 도구가 **“삶 전반에 자연스럽게 깔려 있는 개인 비서”** 에 가까워지도록 만드는 것입니다.

### 사용 방법

1. Python 3가 설치되어 있어야 합니다.
2. 터미널에서 프로젝트 폴더로 이동합니다.

```bash
cd /Users/hyunseung/dev/Ai_helper
```

3. 스크립트를 실행합니다.

```bash
python ai_helper.py
```

4. 화면에 나오는 안내에 따라, 지금 줄이고 싶은 삶의 영역 번호를 선택합니다.
5. 해당 영역에서 자주 발생하는, “실수하면 스트레스만 있고 보상은 적은” 작업 리스트를 보여줍니다.
6. 본인에게 해당되는 항목에 표시해 두고, 체크리스트/캘린더/자동화 대상으로 삼으면 됩니다.

### 기능

- 집/생활, 업무/공부, 금융/행정, 건강/루틴, 디지털 정리 5개 영역 중 하나를 고릅니다.
- 각 영역에서 대표적인 루틴·실수 유발 작업 예시들을 5개 정도 보여줍니다.
- 후속으로, 이 항목들을 어떻게 “체크리스트, 리마인더, 자동화” 후보로 삼을지 간단한 가이드를 제공합니다.

앞으로 기능을 늘리고 싶다면, 예를 들어:

- 사용자가 직접 항목을 추가/수정/저장할 수 있게 만들기
- “자동화 아이디어”를 조금 더 구체적인 템플릿으로 제안하기
- 하루/주간 리뷰 모드 만들어서, 자주 까먹는 일들을 모아 보여주기

같은 방향으로 확장해 나갈 수 있습니다.
